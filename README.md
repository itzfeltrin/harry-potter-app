### Step-by-Step Instructions

1. **Clone the Project:**
  - Open a terminal or command prompt and run:
  ```bash
  git clone git@github.com:itzfeltrin/harry-potter-app.git
  ```
2. **Navigate into the project folder**
  ```bash
  cd harry-potter-app
  ```
3. **Install the dependencies**
  ```bash
  npm run install
  ```
4. **Run dev server**
  ```bash
  npm run dev
  ```
